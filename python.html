Python 介绍

什么是 Python？

Python 是一种高级、解释型、通用型编程语言。它的设计哲学强调代码的可读性和简洁的语法，这使得它成为初学者和专业开发者的理想选择。

基本语法


# 这是一个注释
print("Hello, World!")  # 打印语句

# 变量赋值
x = 5
y = "John"
print(x)
print(y)

# 条件语句
if x > 2:
    print("x 大于 2")
elif x == 2:
    print("x 等于 2")
else:
    print("x 小于 2")

循环


# for 循环
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

# while 循环
i = 1
while i < 6:
    print(i)
    i += 1

读写文件


# 写入文件
with open("example.txt", "w") as file:
    file.write("Hello, World!")

# 读取文件
with open("example.txt", "r") as file:
    content = file.read()
    print(content)

爬虫项目案例


import requests
from bs4 import BeautifulSoup

url = "https://www.example.com"
response = requests.get(url)
soup = BeautifulSoup(response.text, 'html.parser')

# 找到所有的标题元素
titles = soup.find_all('h2')

for title in titles:
    print(title.text)
